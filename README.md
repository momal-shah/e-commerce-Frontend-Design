# e-commerce-Frontend-Design
E-Commerce Web Application (Frontend) A fully responsive, client-side E-Commerce web application built from scratch using HTML5, CSS3, and JavaScript. This project simulates a complete shopping experience, from product discovery to checkout, without relying on backend frameworks.
🚀 Key Features Dynamic Content Rendering: Products are rendered dynamically on the Product Listing and Detail pages using a JavaScript-based data object, simulating a database fetch. State Management with LocalStorage: Persistent Shopping Cart: Items remain in the cart even after closing the browser. Wishlist/Save for Later: Users can move items between cart and wishlist. User Profile data persistence. Advanced Filtering System: Real-time filtering on the listing page by Brand, Price Range, Condition, and Rating. Responsive Design: Fully adaptive layout using CSS Grid and Flexbox, ensuring a seamless experience across Mobile, Tablet, and Desktop devices. Interactive Cart Logic: Quantity adjustment, dynamic price calculation (Tax/Discount), and coupon code validation. 🛠️ Technologies Used HTML5: Semantic markup for accessibility and SEO optimization. CSS3: Custom CSS Variables for theming. CSS Grid & Flexbox for complex layouts. Media Queries for responsiveness. Hover effects and transitions. JavaScript (ES6+): DOM Manipulation. Event Handling. JSON handling for LocalStorage. URL Parameter handling for page routing.
